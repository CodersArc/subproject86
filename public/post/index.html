Nickname Registration Your nick is how people on freenode know you. If you register it, you’ll be able to use the same nick over and over. If you don’t register, someone else may end up registering the nick you want. If you register and use the same nick, people will begin to know you by reputation. If a channel is set to mode +r, you won’t be able to join it unless you are registered and identified to NickServ.
